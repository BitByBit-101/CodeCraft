⚙️ **CodeCraft** – Bhumika's Dev Portfolio

Welcome to my interactive portfolio, a fusion of clean code and creative design.  
Built with **React**, **Framer Motion**, **GSAP**, and 3D elements using **Spline**, this portfolio is more than just a site — it's an experience. ✨


🚀 **Tech Stack**

- ⚛️ **React** + **Vite** – Fast SPA development
- 🎨 **Tailwind CSS** – Utility-first responsive design
- 🎬 **Framer Motion** – Sleek and dynamic animations
- 🌀 **GSAP** – Scroll-triggered and timeline animations
- 🧊 **Spline Tool** – 3D scenes and interactivity
- 🛠️ **Other Tools** – React Icons, React Router, etc.


💻 **Local Setup**

```bash
git clone https://github.com/BitByBit-101/CodeCraft.git
cd CodeCraft
npm install
npm run dev

Make sure Node.js and npm are installed on your system.


🎯 **Features**
🖼️ Scroll-triggered motion effects using GSAP

🧊 Interactive 3D sections powered by Spline

⚡ Smooth animations using Framer Motion

🎨 Tailwind for fast and elegant UI

🌗 Optional dark mode toggle

📱 Fully responsive
